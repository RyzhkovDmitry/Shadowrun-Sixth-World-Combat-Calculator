# Shadowrun 6E Combat Tracker

A streamlined, mobile-friendly web calculator designed to handle the complex combat math of Shadowrun 6th World. This tool helps Game Masters and players resolve attacks, calculate Edge, and track damage soak in seconds.

## Features
- **Edge Detection**: Automatically compares AR and DR to determine Edge gain.
- **Combat Resolution**: Calculates net hits and potential damage.
- **Glitch Tracker**: Identifies standard and critical glitches based on dice pool results.
- **Soak Calculator**: Quickly applies Body-based damage reduction to find final damage.
- **Zero Setup**: A single-file HTML/React application that works directly in any browser via GitHub Pages.

## Tech Stack
- **React** (via CDN)
- **Tailwind CSS** (for cyberpunk-themed UI)
- **Babel** (standalone)

## 🤖 AI-Powered Development
This project was built using **AI-assisted development (Vibe Coding)**. The core logic and UI components were generated and refined through collaborative sessions with **Google Gemini**, utilizing the **Continue** extension in VS Code. 

By leveraging LLMs, the transition from a Swift-based prototype to this functional web application was completed with high efficiency and code accuracy.

## How to Use
1. Enter Attacker's AR and Base Damage.
2. Enter Defender's DR and Body.
3. Input dice roll results (Pool, Hits, Ones).
4. Click **Calculate** to see the result and apply Soak.
